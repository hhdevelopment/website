After launching the application, `ARCHWAY` will self-initialize.

But it requires configuring the `ARCHWAY` administrator.

![setup](img/setup.png)

Enter the name, email and password for the `ARCHWAY` administrator.

:::info
During initialization, an organization named `MAIN` is created with 5 predefined groups.
This is for convenience. Rename the organization and groups as you see fit.
:::
